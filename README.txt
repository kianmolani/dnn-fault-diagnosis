Predictive_Health_Monitoring_Using-Transfer_Learning
Machine learning project for anomaly detection on bearings where the source domain is trained using CRWU dataset and target domain is trained on data from a novel dataset developed in one of the Labs in UofM Robotics Lab.

[Case Western Reserve University Bearing Dataset](https://engineering.case.edu/bearingdatacenter/) dataset
[Processed Markov Transition Field Case Western Reserve University Bearing Dataset](https://drive.google.com/drive/folders/1-2TdZIYEshB5vcXDeB81t4lLZd9fOp9A?usp=sharing/) dataset


[IMS Bearing Dataset](https://www.kaggle.com/datasets/vinayak123tyagi/bearing-dataset/) dataset
[Processed Markov Transition Field IMS Bearing Dataset](https://drive.google.com/drive/folders/14Fy0nGzUINauMWqLCH4vZ79yf8cW69V9?usp=sharing) dataset

Download [Raw and Processed Laboratory Data](https://drive.google.com/drive/u/0/folders/10qEdVO39-jP2uQ3qXL9Ua1cK4IMcOj5k/) dataset

Best Trained Models: https://drive.google.com/drive/folders/1HWbnbc573Uk-PICWnhSsFVck9II-VCWa?usp=share_link
